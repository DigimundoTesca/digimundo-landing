Mobile Cloud Kitchen

MOBILE Cloud Kitchen es el complemento ideal a Cloud Kitchen, para realizar entregas a domicilio.

Cuenta con una completa integración con el sistema en la nube, y sus funcionalidades se adaptan a necesidades específicas.

Mobile CK, cuenta con las siguientes características:

  • Conección con la base de datos de Cloud Kitchen.
  • Mapa que muestra la ruta más rápida para realizar la entrega.
  • Sistema para mostrar pedidos pendientes de forma más ágil y sin complicaciones.
  • Sistema para calificar al repartidor una vez completada la entrega.
